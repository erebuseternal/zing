Conditionals

What are Conditionals for?

Conditionals allow you to have code that only gets executed when a certain condition is satisfied. This means that you can make your code act differently under different conditions.

How are Conditionals Constructed?

Conditionals are constructed from a set of components written one after another. Each component is composed of:

  1. A keyword.
    2. (This allows python to identify the conditional and its components)
  2. a python expression that acts as a condition. (Remember every expression evaluates to True or False [from the previous section on expressions and booleans])
  3. code that should be run if the condition (expression) is true.

How do Conditionals Work?

When your code is run, python will look through each component in order and check its condition. Once it finds a condition that is true, it will execute that component's code and then skip the rest of the components. So the first component who's condition is the only one that has it's code run. This is useful if you have many pieces of code for completely different situations and only want one to execute per situation.

Rules for Writing Conditionals

Keywords

  • First component has an if
  • Subsequent components have elif

Conditions

Any expression works for a condition.

Code

The code should be indented and you can use loops, expressions and/or more conditionals. Other than that it is completely up to you!

Optional else

If you have multiple components, your final one can be a simple else statement:

else:
 some code

Writing Conditionals

The Syntax

The components in a conditional are written in the following way:

keword condition:
 some code

And each of these components follow each other with no gaps in between them. Like:

keyword1 condition1:
 some code
keyword2 condition2:
 some code

The Keywords

Here are the rules:

  • The keyword for the first component is "if".
  • The keyword for the rest of the components is "elif".

So, for example, the above would become:

if condition1:
 some code
elif condition2:
 some code
